# Student-Performance-Analyzer-
The Student Performance Analyzer is a data-driven analytics system that helps educational institutions monitor, evaluate, and predict student academic performance. It combines Machine Learning, Data Analytics, and Interactive Dashboards to identify at-risk students, analyze attendance impact, and provide personalized improvement recommendations.

The project uses historical student data such as attendance records and subject-wise marks to predict whether a student is likely to pass or fail. It also highlights weak subjects, measures performance trends, and assists educators in making informed decisions for student success.
